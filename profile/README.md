# Zero-Cost Labs

**Building open, community-driven platforms for data, compute, and IT — transparent by design, optimized by default.**

Zero-Cost Labs is an open organization dedicated to delivering production-ready, reusable infrastructure platforms. Every component is designed for maximum transparency, built on proven open-source tools, and shared freely with the community. A live ensemble demo of all capabilities is hosted in the [`zcl`](https://github.com/zero-cost-labs/zcl) repository. Full documentation is published and served on [GitHub Pages](https://zero-cost-labs.github.io).

---

## Platforms

### 🔍 [`zcs`](https://github.com/zero-cost-labs/zcs) — Zero-Cost Search
An Elasticsearch-backed search platform providing:
- Robust, reproducible deployment procedures
- Communal index definitions ready for re-use

### 📊 [`zcm`](https://github.com/zero-cost-labs/zcm) — Zero-Cost Monitoring
A Grafana-backed monitoring platform providing:
- Robust, reproducible deployment procedures
- Common settings and data source definitions
- Communal dashboard definitions

### ☸️ [`zcc`](https://github.com/zero-cost-labs/zcc) — Zero-Cost Cluster
A Kubernetes cluster platform built on [`k0s`](https://k0sproject.io), providing:
- Secure, multi-node cluster definitions and deployment procedures
- GitHub Actions integration for zero-cost CI/CD processing
- Self-hosted GitHub Actions Controller for larger workloads
- Monitoring integration via `zcm` (Kubernetes definitions and instructions included)

---

## Philosophy

> Open infrastructure should be accessible, reproducible, and community-owned.

Zero-Cost Labs follows current software trends — immutable infrastructure, GitOps, observability-first — and keeps everything in the open so that every deployment decision can be understood, challenged, and improved by anyone.
